## アプリ概要
シンプルなTODOリストです。  
ReactHooks + Next.jsの制作実践として作成しました。  
アプリURL: https://todo-next-git-main.kazu150.vercel.app/  

## アプリ仕様
- TODOの新規作成、修正、削除、一括削除に対応
- TODOのステータスによって、スタイルを変更
- TODOのソートに対応(タイトル、期限、作成日、最終更新日、状態の昇順降順)
- 書き込んだデータは、ブラウザのlocalStorageにて管理

## 利用ツール
- React + React Hooks
- Vercel
- Next.js
- TypeScript
- Material-ui
